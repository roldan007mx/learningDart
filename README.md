# `exercices_list_processing.dart`

Open this file in your editor. Fill in the five functions. The `main` at the bottom runs tests and prints **Expected** and **Actual** so you can see if it works.

**Basics:** save your file after each change, then run it again in the terminal with the same command.

## Terminal — useful commands

| Command | What it does |
|--------|----------------|
| `cat "file.name"` |  print the content of the file |
| `cd "path/to/folder"` | Moves into that folder (use quotes if the path has spaces). |
| `ls` | Lists files in the current folder. |
| `dart run exercices_list_processing.dart` | Runs your script (run this from the folder that contains the file). |
